Supprimer mon compte

Vous avez la possibilité de supprimer votre compte Dentis et les données associées. Cette action n’aura pas automatiquement pour effet de supprimer vos données personnelles des bases de données des soignants avec lesquels vous avez pris rendez-vous et/ou que vous avez consulté. Les soignants peuvent avoir un intérêt légitime à les conserver.
